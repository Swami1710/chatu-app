# 🚀 Chatu - Real Time Chat App

A modern chat application built using Flutter & Firebase with real-time messaging and premium UI.

---

## ✨ Features
- 🔐 Google Authentication
- 💬 Real-time messaging
- 👀 Seen & typing indicators
- 🖼️ Image sharing (Cloudinary)
- 🎨 Premium UI (Glass + Gradient)
- 🔔 Push Notifications (FCM)

---

## 🛠️ Tech Stack
- Flutter
- Firebase Auth
- Firestore
- Firebase Messaging
- Cloudinary
